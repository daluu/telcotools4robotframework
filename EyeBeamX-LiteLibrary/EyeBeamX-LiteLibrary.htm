Documentation Resource file of custom user keywords acting as library for eyeBeam/X-Lite softphone automation. Requires use of 3rd party AutoIt library (http://code.google.com/p/robotframework-autoitlibrary/), and the standard Operating System library. For use with eyeBeam version 1.1 or 1.5 (possible support for other 1.x versions), and X-Lite 3.0. The library can be modified to support other softphones, including the latest CounterPath softphones, Bria and X-Lite 4.0 (http://www.counterpath.com/x-lite.html). *NOTE:* Designed to run locally on RobotFramework, controlling softphone on same machine, but can be adapted to run against remote machines running the softphone using the Remote Library (implemented/imported with AutoIt and Operating System libraries). *NOTE:* Not all keywords fully tested, there might be missed bugs or typos, but should mostly work in general. Please file any bugs you find. This library is a port of a VBScript command line script with same API. For how to modify/improve the library, or fix bugs, see the Robot Framework user guide documentation on resource files (http://robotframework.googlecode.com/hg/doc/userguide/RobotFrameworkUserGuide.html?r=2.7.5#resource-and-variable-files).

Answer Call [Arguments] ${ringCount}=${1}            
  [Documentation] Answer call after X rings, where rings are calculated as 1 ring ~ 2 seconds.\n\nDefault ring count is 1, if none given.\n\nRing count must be specified as a number in test case, with format being *${Number}*, where Number is 1 or greater, rather than just supplying number without those brackets and dollar sign.            
  Sleep ${ringCount*2} seconds            
  Send {ENTER} # key          
Ban Call [Documentation] Feature is a softkey in eyeBeam (v1.1) display, not available on X-Lite v3, and untested for eyeBeam v1.5+.\n\nHow to un-ban though?            
  Mouse Click LEFT 425 111 1 0 # Left-click mouse @ Ban softkey # Works like "send to voicemail" all the time until un-banned. How to un-ban though?
Call [Arguments] ${dialString}=${EMPTY}            
  [Documentation] Call the number specified by the dial string, or currently dialed digits (shown on the eyebeam/X-Lite display) if dial string not supplied (or empty string).            
  Run Keyword If '${dialString}'!='${EMPTY}' Send ${dialString} #dial number/string      
  Send {ENTER} # key to place call          
Close Softphone [Documentation] Close or exit the softphone application.            
  Send ^q # Ctrl + q          
Dial [Arguments] ${digits}            
  [Documentation] Dial valid DTMF digits 0-9,*, and #.\n\nDoes not establish a call, use Call keyword for that.\n\nTo make a call with Dial keyword, follow with a Call keyword with no arguments.            
  Send ${digits}            
Detect Message Waiting [Documentation] Detect message waiting indicator (MWI).\n\nReturns 1 if found.\n\nNot supported on X-Lite. Untested on eyebeam v1.5. # Detect MWI icon. Left-click mouse @ MWI icon.          
  Mouse Click LEFT 315 144 1 0    
  ${rc}= Win Wait Active Messages Waiting ${EMPTY} 5      
  Run Keyword If '${rc}'=='1' Send {ENTER} # dismiss popup MWI window/message      
  [Return] ${rc}            
Flash [Arguments] ${version}=1.5            
  [Documentation] Press Flash.\n\nIf using eyebeam version 1.1, need to supply argument value of "1.1". Default keyword comand is for eyeBeam version 1.5 and X-Lite.            
  Run Keyword If '${version}'=='1.1' Mouse Click LEFT 300 236 1 0
  Run Keyword If '${version}'=='1.5' Mouse Click LEFT 318 236 1 0
Go Offhook [Documentation] Go offhook (though not literally for a SIP client) to get dial tone & begin dialing.\n\nAssumes no number or SIP URI has been dialed beforehand.\n\nGenerally, you should just use the Call keyword and supply a dial string. Otherwise you can do this sequence of keywords: go offhook, dial digits, call; or go offhook, call dial string.            
  Send {ENTER}

Hold [Documentation] Toggle call hold feature of softphone.\n\nDefault state is to put call on hold when first invoked on active call.            
  Send ^o # Ctrl + o          
Ignore Call [Documentation] Use eyeBeam/X-Lite on-screen display softkey to ignore incoming call. # Works like "send to voicemail"          
  Send ^i # Ctrl + i          
Perform Blind Transfer [Arguments] ${dialString}            
  [Documentation] Perform a blind transfer of current call to specified destination defined by dial string.\n\nNot supported on X-Lite.\n\nIdeally should check display area to see if get message that transfer failed or not, which currently is not provided by this library.            
  Send ^t # Ctrl + t to initiate xfer          
  Send ${dialString} # dial destination of xfer          
  Send ^t # Ctrl + t to complete xfer          
Perform Consulted Transfer [Arguments] ${line}            
  [Documentation] Perform a consulted or attended transfer of current call to another line (1-6).\n\nNot supported on X-Lite.\n\nIdeally should check display area to see if get message that transfer failed or not, which currently is not provided by this library.            
  Send ^t # Ctrl + t to initiate xfer          
  Select Line ${line} # that is currently on hold          
  Send ^t # Ctrl + t to complete xfer          
Redial [Documentation] Redial the last dialed number.            
  Send ^r # Ctrl + r          
Select Line [Arguments] ${line}=1            
  [Documentation] Select a line from 1 to 6 on eyeBeam or 1 to 2 on X-Lite.\n\nDefaults to line 1.            
  Send ^${line} # Ctrl + line number          
Send Audio From Softphone [Arguments] ${audioFile}            
  [Documentation] Plays specified wave audio file as audio sent out from eyebeam/X-Lite softphone.\n\nThis keyword assumes that on the computer running the softphone, the audio output is looped back to audio line input or microphone input to be transmitted to the other end of the SIP call, or the audio is piped from wave audio player to softphone audio input. The audio output to input can be piped or looped via virtual audio driver software, 3.5mm loopback audio cable connecting audio out to audio in, or the sound card driver software's built in audio mixer feature.\n\nThe audio playback is done through Windows Media Player, version 6.x. Windows Media Player will be launched to play the file and then auto-close on completion. Alternatively, can be reconfigured to use a different audio playback application like Windows Sound Recorder, etc.            
  Run "C:\Program Files\Windows Media Player\mplayer2.exe" /play /close ${audioFile} #other command options include "mplay32 /play /close ${audioFile}" and "sndrec32 /play /close ${audioFile}"          
Send To Voicemail [Documentation] Send incoming call directly to the voicemail of the eyeBeam/X-Lite SIP extension/account, if offered/supported by SIP service provider.\n\nAssumes voicemail settings are pre-configured on softphone.\n\nAlternatively, it may be simpler to use the Ignore Call keyword to do the same thing, since that doesn't require voicemail configuration on the softphone.            
  Send ^s # Ctrl + s          
Set Focus To Softphone [Documentation] Bring softphone into focus as active window on desktop for running RobotFramework test automation against it.\n\nUse as test setup precondition whenever executing the first of a set of softphone automation commands.            
  Win Activate eyeBeam ${EMPTY} # if using eyeBeam        
  Win Activate X-Lite ${EMPTY} # if using X-Lite        
  Auto It Set Option MouseCoordMode 0 #set relative mouse positioning for softphone automation
